The Humain Kiosk C library (`kiosk-c`) is a C99 client for devices that cannot run a full
operating system or browser runtime. It wraps the [Device REST API](/connections/rest-text) and
[WebSocket streaming API](/connections/websocket-streaming) in a minimal, dependency-light interface
backed by **libcurl**.

All AI processing is server-side. The library sends text or audio to the Humain backend and
receives responses — the device itself needs no LLM, STT, or TTS capability.

---

## Requirements

| Requirement | Minimum |
|---|---|
| C standard | C99 |
| libcurl | 7.68.0 |
| TLS | OpenSSL 1.1+ or mbedTLS 2.16+ |
| RAM (text sessions) | 64 KB available heap |
| RAM (voice sessions) | 512 KB available heap + audio buffers |
| Flash / storage | ~180 KB for the static library |
| Network | TCP/IP with DNS, HTTPS (port 443) |

<Note>
  Voice sessions require audio capture hardware and ALSA or PulseAudio on Linux. FreeRTOS and
  Zephyr targets support **text and streaming only** — see the platform guides for details.
</Note>
