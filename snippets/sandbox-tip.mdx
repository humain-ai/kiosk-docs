<Tip>
  During development, issue an **`hk_test_`** credential from the admin panel instead of a live
  `hk_live_` one. Sandbox is a property of the credential, not a different base URL — you call
  the same `https://api.humain.ai` host either way. Sandbox sessions don't consume production
  LLM credits and aren't billed. See [Sandbox mode](/concepts/sandbox-mode).
</Tip>
