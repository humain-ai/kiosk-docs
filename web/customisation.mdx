The Humain widget ships with a complete design system expressed as CSS custom properties. You can
theme it with a few variable overrides, supply a full theme object, or replace individual components
entirely with your own.

---

### Quick brand colour override

```css
/* Apply to :root or a container element */
:root {
  --hk-primary:       #7c3aed; /* your brand colour */
  --hk-primary-hover: #5b21b6;
}
```

## Custom themes

For larger-scale customisation, pass a `theme` object to `HumainWidget.init()`, `<KioskChat>`, or
`HumainKioskClient`:

```javascript
HumainWidget.init({
  credential: "hk_live_your_credential_here",
  theme: {
    primary:         "#7c3aed",
    primaryHover:    "#5b21b6",
    background:      "#ffffff",
    surface:         "#f5f3ff",
    border:          "#ede9fe",
    text:            "#1e1b4b",
    textMuted:       "#6b7280",
    radius:          "8px",
    fabSize:         "60px",
    width:           "400px",
    dark: {
      // Override any property specifically in dark mode
      background: "#1e1b4b",
      surface:    "#312e81",
    },
  },
})
```

```tsx
// React
<KioskChat
  credential="hk_live_your_credential_here"
  theme={{
    primary: "#7c3aed",
    radius: "8px",
  }}
/>
```

```vue
// Vue
<KioskChat
  credential="hk_live_your_credential_here"
  :theme="{ primary: '#7c3aed', radius: '8px' }"
/>
```

## Custom components

Replace individual internal components with your own. This uses a render prop / slot pattern that
lets you keep the widget's state machine while controlling the visual output.

### React — render props

```tsx
import { KioskChat, type MessageBubbleProps } from "@humain/kiosk-js/react"

// Custom message bubble
function MyBubble({ role, content, streaming }: MessageBubbleProps) {
  return (
    <div className={`my-bubble my-bubble--${role}`}>
      <span className="my-bubble__avatar">{role === "user" ? "You" : "AI"}</span>
      <p className={streaming ? "my-bubble__text--streaming" : "my-bubble__text"}>
        {content}
      </p>
    </div>
  )
}

export function Chat() {
  return (
    <KioskChat
      credential="hk_live_your_credential_here"
      components={{
        MessageBubble: MyBubble,
        // Also replaceable: Header, InputArea, SendButton, TypingIndicator
      }}
    />
  )
}
```

### Vue — slots

```vue
<template>
  <KioskChat credential="hk_live_your_credential_here">
    <template #message="{ role, content, streaming }">
      <div :class="['my-bubble', `my-bubble--${role}`]">
        <span>{{ role === 'user' ? 'You' : 'AI' }}</span>
        <p :class="{ streaming }">{{ content }}</p>
      </div>
    </template>

    <template #header>
      <div class="my-header">
        <img src="/logo.svg" alt="Logo" />
        <h2>Support</h2>
      </div>
    </template>
  </KioskChat>
</template>
```

### Available slots / render props

| Name | Props / slot data | What it replaces |
|---|---|---|
| `MessageBubble` | `role`, `content`, `streaming` | Each message in the history list |
| `Header` | `title`, `onClose` | The coloured header bar |
| `InputArea` | `value`, `onChange`, `onSubmit`, `disabled` | The input field and send button row |
| `SendButton` | `disabled`, `onClick` | Just the send button |
| `TypingIndicator` | — | The `…` animation while waiting for the first token |
| `EmptyState` | — | The content shown before the first message |
